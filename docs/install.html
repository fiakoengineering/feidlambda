Panduan ini untuk Advanced Formula Environment v1.1.

1 Advanced Formula Environment

Install add-ins Advanced Formula Environment dari menu Insert > Get Add-ins > Cari Advanced Formula Environment (AFE).

2 Import menggunakan AFE

Setelah AFE terpasang di Microsoft Excel, lakukan import dengan membuka AFE melalui Formula > Advanced Formula Environment > Import.

3 Salin Github Gist URL

Isi “Github Gist URL” dengan link koleksi lambda yang tersedia. Checklist ✅ bagian “Add formulas to new module” dan isi nama module dengan nama koleksi (contoh: feid). Berikut link feidlambda v0.3.x:

https://gist.github.com/taruma/92bd33600a3d42dc9aead87558404a12

4 Save / Sync module

Setelah import seluruh koleksi yang digunakan dan berhasil, lakukan saving (sync) yang bertujuan untuk menerapkan module ke lembar kerja Excel. Shortcut untuk save yaitu Ctrl + S.

5 Gunakan koleksi lambda

Selesai save, fungsi feidlambda bisa langsung digunakan.

6 Baca dokumentasi feidlambda v0.3

Dokumentasi feidlambda v0.3 bisa baca di halaman feidlambda v0.3. Dan panduan migrasi bisa baca di halaman Migrasi feidlambda v0.2 ke v0.3.
